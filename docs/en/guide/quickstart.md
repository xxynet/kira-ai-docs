# Quick Start

KiraAI supports multiple deployment methods, including but not limited to:

- [Windows Script Startup](/en/deployment/windows)
- [Linux Script Startup](/en/deployment/linux)
- [Docker Container Deployment](/en/deployment/docker)

## Configuration

After deploying KiraAI according to the deployment tutorials above, you can configure it through the WebUI after starting the project:

1. Access the Web management interface
2. Configure the following content:
   - Provider (LLM service provider)
   - Adapter (chat platform connection)
   - Personality (personality settings for the digital life)
   - Other advanced configurations
