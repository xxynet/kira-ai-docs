# 快速开始

KiraAI 支持多种部署方式，包括但不限于：

- [Windows 脚本启动](/deployment/windows)
- [Linux 脚本启动](/deployment/linux)
- [Docker 容器化部署](/deployment/docker)

## 配置

根据上面的部署教程部署KiraAI，启动项目后，你可以通过WebUI进行配置：

1. 访问Web管理界面
2. 配置以下内容：
   - 提供商（LLM服务提供商）
   - 适配器（聊天平台连接）
   - 人设（数字生命的个性设定）
   - 其他高级配置
